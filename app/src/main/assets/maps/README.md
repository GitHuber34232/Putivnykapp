# Офлайн карти для Putivnyk

Коротко: у проєкті потрібен файл карти `kyiv.map` у папці `app/src/main/assets/maps/`.

Я вже підготував(ла) готові карти — якщо вони у вас є, просто покладіть `kyiv.map` у згадану папку.

Якщо хочете робити картографічний pipeline самостійно, у проєкті є скрипти в `scripts/` з інструкціями — користуйтеся ними на власний розсуд. Я не додаю тут довгі команди, щоб README був компактним.

Порада: розмір `kyiv.map` зазвичай 20–60 МБ; перевірте, щоб файл не мав обмежень доступу.
